PSV 3.65到3.68系统破解教程

发表于 2018-07-07 | 分类于 游戏 |

事前准备

一台 登陆了账号，且系统版本为3.65 到 3.68 系统的 psv

一台能够使用的电脑

仅此而已

开始破解

下载破解所需的东西

下载
h-encore,
qcma,
psvimgtools,
pkg2zip,
bitter smile(游戏)

解压破解文件

安装 qcma

把 h-encore 解压出来，psvimgtools 解压到 h-encore 文件夹里面

pkg2zip 和 bitter smile 的 pkg 放在一起

准备破解使用的软件

使用cmd，切换到pkg2zip的目录后执行 pkg2zip -x xGMrXOkORxWRyqzLMihZPqsXAbAXLzvAdJFqtPJLAZTgOcqJobxQAhLNbgiFydVlcmVOrpZKklOYxizQCRpiLfjeROuWivGXfwgkq.pkg

完成后会多出一个 app\PCSG90096 的文件夹

把 PCSG90096 这个文件夹里面的东西丢进 h-encore\app\ux0_temp_game_PCSG90096_app_PCSG90096 里面 (也就是 eboot.bin 和 VITA_PATH.TXT 放在一起)

把 app/PCSG90096/sce_sys/package/temp.bin 这个文件丢到 h-encore/license/ux0_temp_game_PCSG90096_license_app_PCSG90096 里面，并改名为 6488b73b912a753a492e2714e9b38bc7.rif 。此处需注意，后缀名是.rif，而不是.bin

打开 qcma，并在设置里把 Use this version for updates 改成 FW 0.00 (Always up-to-date)

打开你的 psv，并使用 usb 线连接到电脑。打开内容管理，选择复制内容，个人电脑→PS Vita

这时候你电脑文档中会出现一个 PS Vita/APP/xxxxxxxxxxxxxxxx 的文件夹，把文件夹名复制出来，输入进这个网页，它会给你另外一串key

在 h-encore 下建立一个 bat 文件，内容如下

YOUR_KEY 均替换成你刚刚获取到的一长串 key

直接运行这个bat即可

安装破解程序到psv上

把 h-encore/PCSG90096 放进文档中的 PS Vita/APP/xxxxxxxxxxxxxxxx/PCSG90096

右键 qcma，选择 Refresh database

在 psv 上打开内容管理，选择复制内容，个人电脑→PS Vita，应用程式，PS Vita

直接安装 h-encore

在 psv 上打开 h-encore 程序，选择 install henkaku 和 download VitaShell

以后重启的时候只需要选择 install henkaku 就可以了

问题解决

破解问题

  • 我打开 h-encore 的时候，它一直白屏 - 这个问题有的时候会出现，你只需要按住电源键强制重启，再次开机进行破解应该就可以了

  • 打开 h-encore 的时候看到 C2-12828-1 错误 - 这是极其少见的错误，重新打开破解就好了

  • 当我打开 h-encore 的时候，它运行了 bitter smile demo 这款游戏 - 你的存档出问题了，从头重新安装破解即可

来源

h-encore GitHub 仓库
